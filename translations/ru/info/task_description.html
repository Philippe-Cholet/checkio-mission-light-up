Эта миссия является адаптацией игры "Освещение" (из коллекции головоломок Саймона Тэтхема). Если вы не знакомы с этой игрой, или просто хотите поиграть, то вам сюда.

Вы находитесь на прямоугольной сетке и должны осветить все клетки. Для этого вам нужно поместить несколько источников света (лампочек) в темные клетки, при этом ни одна лампочка не должна освещать другую. Лампочки светят в четырех направлениях от стены до стены. Возле стены, представленной числом, должно быть расположено соответствующее количество лампочек.

Сетка будет представлена набором строк. Клетки, находящиеся в темноте, будут представлены пробелом, а стены - симоволом 'X' или числами '0', '1', '2', '3' and '4'.

Вам предстоит вернуть координаты всех лампочек на сетке, которые смогут осветить все темные клетки. 



Возможное расположение огней: [(0, 5), (5, 6), (3, 3), (4, 4), (0, 0), (2, 2),
                               (6, 5), (6, 1), (5, 0), (1, 1), (1, 6)]

('    1  ',
 '   0X  ',
 'X2     ',
 ' 0   0 ',
 '     11',
 '  X0   ',
 '  1    ')

Входные данные: Кортеж, содержащий строки.

Выходные данные: Последовательность (iterable), содержащая координаты (кортежи/списки, содержащие два целых числа).

Пример: 

list(checkio(('    1  ',
              '   0X  ',
              'X2     ',
              ' 0   0 ',
              '     11',
              '  X0   ',
              '  1    '))) = [(0, 5), (5, 6), (3, 3), (4, 4), (0, 0), (2, 2),
                              (6, 5), (6, 1), (5, 0), (1, 1), (1, 6)]

Предусловия:

  • Все предоставленные головоломки имеют решение.
  • 7 ≤ len(grid) ≤ 50 and 7 ≤ len(grid[0]) ≤ 60.
  • all(len(row) == len(grid[0]) for row in grid).
